A WooCommerce plugin to help customers find their fit by entering height, weight, and typical size.

== Description ==

Find Your Fit is a WooCommerce plugin that helps customers determine the appropriate product size based on their height, weight, and usual size. The plugin offers an interactive form on product pages where users can input their measurements and receive a size recommendation, enhancing the shopping experience and reducing size-related returns.

The plugin provides customization options, including:
* The ability to hide or show the button on non-variable products.
* The ability to adjust the button style with various pre-defined designs.
* The ability to select where the button should appear on the product page.
* AJAX-powered size recommendation logic.

Key Features:
* Customize the placement of the "Find Your Fit" button on product pages.
* Choose from multiple button styles to match your store's design.
* Hide or show the button for specific categories or non-variable products.
* Localize and customize the size recommendation messages.
* AJAX-powered size suggestion based on customer input.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/find-your-fit` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to **Find Your Fit** settings from the WordPress admin dashboard to configure the plugin.
4. Adjust the size recommendation settings based on your needs, select button styles, and more.
5. The button will appear on your WooCommerce product pages based on your settings.

== Frequently Asked Questions ==

= How does the size recommendation work? =
The plugin uses the customer's input (height, weight, and typical size) to provide a size recommendation. You can customize the logic within the plugin's code to fit your store's specific needs.

= Can I hide the button for certain product categories? =
Yes, you can hide the "Find Your Fit" button for specific WooCommerce categories from the plugin's settings page.

= Is this plugin compatible with all themes? =
Yes, Find Your Fit is designed to work with any theme that supports WooCommerce. However, for custom themes, slight styling adjustments may be required.

= Can I customize the button's appearance? =
Yes, the plugin provides multiple pre-defined button styles. You can choose the one that fits your store's design from the settings page.

= Does the plugin support variable products? =
Yes, Find Your Fit works with both simple and variable products. The button can be configured to show or hide based on the product type.
